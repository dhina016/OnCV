STEPS
About *

This is details about you like Area of Interest, Objective etc... About added only one time then you can only update

Education *

This is about your education and experience. If you don't have experience then leave it. This can accept multiple times so you can add school and college details. Postion is where you want to show your details. Eg postion 1 - School details, Postion = 2 College Details. Then in website 1st school details will be shown and then college. Position for formatting the details

Skill *

Skill Category for styling. Name = type of your skill like Technical,Language etc. You can pick your own syle like dotted, line etc. If any doubt check out my skills. Once add the category then you can add your skills. Icon Can give more styling. Select category then add your skill with percentage

Hobbies *

Add name and pick icon which is apt for your Hobbies. You can edit it anytime

Portfolio

Showcase your talent with proof. Add category like drawing, editing, projects etc.. Then go to portfolio and photo with optional description.

Social Links

Share your profile with this social links.

CV Design

Create your own design. Pick color, theme, animation. Status will be set active only if you add all * fields. If you want to check without publish set status to editing. once status is yes the resume will be set public

Tips

Tick style in skill dont have percentage. So you will use it for strength and weakness. tick = strength, cross = weakness, tick symbol is default and 5 is for cross. Anytime you can view instruction. this is available on top!!. Hope you will like this
